# Dotfiles

## How to use

### the actual user needs to an administrator
```
bash <(curl -L s.codingtino.com/config)
```


## what it does

1. installs Xcode
2. installs Nix-Package-Manager
3. clones this Repository to ~/.config
4. installs nix-darwin using the ~/.config/nix-darwin/flake.nix
